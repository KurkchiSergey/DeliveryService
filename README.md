⚠️ Эта задача — усложнение первой задачи, поэтому делается в том же проекте. В качестве домашнего задания вы сдаёте один проект, в котором первая или вторая задача.

Добавьте, кроме общей суммы, вывод информации о том, в какое количество уникальных стран были оформлены доставки. Уникальность означает, что если по России было отправлено, например, три заказа, то учитываться это должно как одна страна.
